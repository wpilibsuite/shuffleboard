# shuffleboard

## Running

To run shuffleboard use the command `./gradlew run`.

### Requirements
- [JDK 8](http://www.oracle.com/technetwork/java/javase/downloads/index.html)

## Building

To build shuffleboard use the command `./gradlew shadowJar`.

### Requirements
- [JDK 8](http://www.oracle.com/technetwork/java/javase/downloads/index.html)
